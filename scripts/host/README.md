# Host Scripts

This folder is for notes and scripts that run on the Windows host machine.

Most GhostGate scripts run inside the Linux Router VM. VMware Workstation Pro VM
creation is usually safest through the graphical UI because ISO paths, storage
locations, and VM hardware settings vary by machine.

## Current File

| File | Purpose |
|---|---|
| `vmware-notes.md` | Records the correct VMware Day 2 adapter layout |

## Safety

Use VMware Workstation Pro to create the VMs and set adapters exactly as
documented in `docs/vmware-setup.md`.
