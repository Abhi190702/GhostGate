# Ethical Use Policy

GhostGate is an educational networking and privacy engineering project.

It is designed to help learners understand:

- Linux networking
- DNS
- NAT
- firewalls
- VPN tunnels
- Tor routing
- traffic monitoring
- privacy limitations

## Allowed Use

GhostGate should be used for:

- personal lab experiments
- networking education
- defensive security learning
- privacy research
- open-source learning
- understanding how traffic flows

## Not Allowed

GhostGate should not be used for:

- illegal activity
- unauthorized access
- abuse of networks
- hiding harmful activity
- attacking websites, services, or users
- bypassing systems in harmful ways

## Privacy Reality

GhostGate does not promise perfect anonymity.

Even with VPN or Tor, privacy can still be affected by:

- browser fingerprinting
- account logins
- cookies
- malware
- timing correlation
- DNS leaks
- misconfiguration
- VPS provider logs
- Tor exit node behavior

The project is about understanding traffic, reducing leaks, and learning how
privacy tools work.
